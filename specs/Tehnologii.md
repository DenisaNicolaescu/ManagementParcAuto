# Tehnologii propuse

## Frontend
- HTML pentru structură
- CSS pentru design modern, carduri și layout
- JavaScript pentru navigare, filtru simplu și interactivitate

## Backend
- PHP pentru logica server și procesare formulare
- MySQL pentru baza de date a flotei

## Arhitectură recomandată
- Structură MVC simplă pentru PHP
- Tabele MySQL dedicate pentru mașini, șoferi, service, asigurări, viniete, anvelope
- API intern PHP pentru operațiuni CRUD
- Interfață HTML/JS statică în faza de revizuire / mockup

## Structura bazei de date
- `masini`
- `soferi`
- `service`
- `asigurari`
- `viniete`
- `anvelope`
- `montaje`
